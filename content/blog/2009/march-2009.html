Presentation Information

Scala is a programming language that integrates both functional and object-oriented patterns in a concise and type-safe way.  Scala runs on the JVM and smoothly integrates with Java.  The presentation will assume familiarity with Java and will focus on the differences in Scala that allow interesting programming approaches.
